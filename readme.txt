== Installation ==

1. Upload the 'powerpack-lite' folder to the '/wp-content/plugins/' directory.
2. Activate the "PowerPack Lite" plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= Is PowerPack Free? =

Yes! PowerPack's core features are and always will be free.

= Should I purchase a paid plan? =

PowerPack Pro includes advance features of each modules like 15+ desings for each, Advanced Shortcode Option, Widgets of relative modules, "Visual Composer" page builder suupport and ofcourse extra modules like Login Customizer, Extra Info Sidebar, Business Hour Widget and much more!

If you're interested in advance modules available, checkout our [paid plans](http://powerpack.wponlinesupport.com/#price).

= 3) Does PowerPack work with my WordPress themes? =

Yes! PowerPack works with all WordPress theme created with WordPress standard.

= 4) Does PowerPack slow down my website? =

PowerPack is designed from the ground-up with performance, SEO and efficiency in mind.

Easy enable / disable feature of modules so use only those modules (Element) which is required to you!!!
